# OscJS

A lightweight JavaScript utility library providing:

- Colorful console logging with `cs()`
- DOM helpers: `el()`, `$()`
- Safe prototype extensions via `PTT()`
- Dynamic right-click context menus (`ctxmenu`)
- Function debouncing with `Function.prototype.debounce`
- And more

## Installation

Clone the repository:

```bash
git clone https://github.com/Oscilllate/OscJS.git
cd OscJS
```
Include `osc.js` in your project:
```html
<script src="./javascript/osc.js"></script>
```
